Common Commands for git hub

git init:

This Command will initialize the current folder and add .gitignore file


git add:

This command will add your files in temporary phase


git commit -m "Some message "

This command will commit changes in the repository

>
>

git branch branch_name:

It will create a new branch

>

git checkout branch_name

It will start pointing to new branch

git merge branch_name

This command will merge the branch where you are sitting to main branch
